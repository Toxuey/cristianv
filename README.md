# Sitio profesional - Cristian Villamizar

Sitio web profesional para el perfil de Cristian Villamizar (Gestión de instalaciones). Proyecto estático con foco en claridad, credibilidad y contenido operativo.

## Tecnologías

- Astro
- Tailwind CSS

## Requisitos

- Node.js (LTS recomendado)
- pnpm

## Instalación

```sh
pnpm install
```

## Desarrollo local

```sh
pnpm dev
```

Servidor local: `http://localhost:4321`

## Build

```sh
pnpm build
```

Salida en `dist/`.

## Vista previa del build

```sh
pnpm preview
```

## Estructura

```
/ 
├── public/              # Assets estáticos
├── src/
│   ├── layouts/         # Layout base
│   ├── pages/           # Páginas del sitio
│   └── styles/          # Estilos globales (Tailwind)
└── package.json
```

## Páginas principales

- `/` Home
- `/about` Sobre mí
- `/projects` Proyectos internos
- `/projects/ptw`
- `/projects/help-desk`
- `/projects/report-management`
